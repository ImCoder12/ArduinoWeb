Подключение датчика температуры DS18B20 к Arduino

Средний уровень 25 мин

Необходимые компоненты

  • Arduino Uno
  • Датчик DS18B20
  • Резистор 4.7 кОм
  • Соединительные провода
  • Макетная плата

Схема подключения

Подключите VCC датчика к 5V Arduino

Подключите GND датчика к GND Arduino

Подключите DQ датчика к цифровому пину 2

Добавьте подтягивающий резистор 4.7 кОм между VCC и DQ

Программный код

Установите библиотеку DallasTemperature через менеджер библиотек Arduino IDE:


1. Откройте Arduino IDE
2. Скетч → Подключить библиотеку → Управлять библиотеками...
3. Найдите "DallasTemperature"
4. Установите последнюю версию
                        

#include <OneWire.h>
#include <DallasTemperature.h>

#define ONE_WIRE_BUS 2

OneWire oneWire(ONE_WIRE_BUS);
DallasTemperature sensors(&oneWire);

void setup() {
  Serial.begin(9600);
  sensors.begin();
}

void loop() {
  sensors.requestTemperatures();
  float tempC = sensors.getTempCByIndex(0);
  
  Serial.print("Температура: ");
  Serial.print(tempC);
  Serial.println(" °C");
  
  delay(1000);
}

Решение проблем

Проверьте правильность подключения резистора и целостность соединений. Убедитесь, что выбран правильный пин в коде.

Попробуйте другой датчик, чтобы исключить его неисправность.

Обычно означает обрыв соединения. Проверьте пайку и контакты датчика.

Убедитесь, что подтягивающий резистор подключен правильно.

Проверьте питание датчика - должно быть стабильные 5V.

Убедитесь, что датчик не находится рядом с источниками тепла.
